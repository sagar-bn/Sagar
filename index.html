Role-Based Access Control System

×

🎯 Project Overview

A secure REST API built with Spring Boot, featuring JWT authentication and Role-Based Access Control (RBAC). This system enables granular authorization, allowing different user roles to access specific resources based on their assigned permissions.

🛠️ Technical Implementation

Implemented JWT for stateless authentication, including token generation and validation. Developed RBAC with Many-to-Many mapping between Users and Roles for flexible permission management. Utilized Spring Security with BCrypt password hashing for robust security and custom filters for request processing.

🔧 Technologies Used

Java 21 Spring Boot Spring Security JWT BCrypt MySQL Maven REST API
